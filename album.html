LENS 管理員設定與登入

⚠️ 目前已啟用本地測試模式！
您不需輸入任何 GitHub / Cloudinary 金鑰即可測試所有編輯與拖拉上傳功能。變更將僅儲存在本機瀏覽器中。
格式為：用戶名/儲存庫名
需具備 `contents` 的讀寫權限以同步 `data.json` 檔案
用於上傳與自動優化高解析度照片
需在 Cloudinary 的 Settings -> Upload 中啟用 Unsigned upload

第一步：準備 GitHub 儲存庫

  1. 在 GitHub 建立一個新儲存庫（例如 lens-portfolio），將專案所有檔案推送到該 Repo 上。
  2. 開啟該專案的 GitHub Pages（在 Settings -> Pages），選取 main 分支的 / (root) 並儲存。

第二步：申請 GitHub Token

  1. 登入 GitHub -> 點擊右上角頭像 -> 進入 Settings。
  2. 最下方點擊 Developer Settings -> Personal Access Tokens -> 選擇 Fine-grained tokens。
  3. 點擊 Generate new token，名稱填入 LENS-CMS。
  4. Repository access 選擇 Only select repositories 並選擇您的作品集專案。
  5. 在 Permissions 下의 Repository permissions 中，將 Contents 設為 Read and Write。
  6. 點擊產生，將權杖複製下來並貼在左側登入欄位中。

第三步：設定免費圖床 (Cloudinary)

  1. 註冊並登入免費的 Cloudinary 帳戶。
  2. 在 Dashboard 儀表板頁面複製您的 Cloud Name 並填入左側欄位。
  3. 點擊左下角 Settings（齒輪） -> 進入 Upload 頁面。
  4. 往下滾動找到 Upload presets，點擊 Add upload preset。
  5. 將 Signing Mode 修改為 Unsigned，點擊上方 Save。
  6. 將該 Preset 名稱（通常是一串隨機字元或 ml_default）填入左側對應欄位中即可。
